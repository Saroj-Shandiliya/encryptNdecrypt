About us :)

Freedom and Control over data. We here at encryptNdecrypt focus a lot on your freedom to surf anywhere and everywhere throughout the internet and let security not be a concern for you. :)

Data Privacy of each and every member of the encryptNdecrypt family is what we focus on. :)

We provide peer to peer encrypted messaging.

How it works :)

The messages we encrypt is encrypted in a way it can be decrypted only once. So once message is decrypted the encrypted text remains of no use to anyone and is just a peice of random text. We do not store messages in plain format ,we store encrypted version of those text and once decrypted it becomes of no use.

Modes:

We provide four different modes of messaging for users:

1. A user to user direct messaging mode

2. A user to user key messaging mode (here users need key to decryptthe message)

3. A guest messaging mode(here user can share a special guest id with others and geust id can only be accessed once)

4. A random mode(here users create a random id and share it to a group of people and these random id is valid only till mid night)
